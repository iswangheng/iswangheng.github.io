## 关于 AI Agent 工程实践（2026-02-26）

### Files Over Databases
- 文件是人类和 Agent 都可读的 — 无需中间层
- 调试友好 — 直接 cat 就能看到状态
- 版本控制天然支持 — git 追踪一切变化

### Playbooks Over Fine-Tuning
- 运行时策略，修改立即生效 — 无需重新训练
- 可解释性强 — 知道为什么这么做
- 灵活迭代 — 规则文件随时可改

### Anti-Fragility 设计
- 失败 → 记录 → 规则 → 下次避免
- 每次错误都是进化机会
- 7 天后精简归档，保持系统健康

### 2026 AI Agent 现状
- 40% Agent 项目会在 2027 年前被取消（可靠性问题）
- MCP + A2A = Agent 互操作的 TCP/IP
- LangGraph 是 2026 S 级框架（400+ 公司生产使用）

## 关于 AI 时代

### AI 替代劳动的本质问题
- **不是技术问题，是权力问题** — 谁来决定谁有饭吃？
- **幽灵 GDP** — 产出在账面上，但不流向实体经济
- **没有刹车的反馈循环** — 裁员→消费降→投AI→更多裁员

### 个人安身立命的三层防御
1. **护城河（短期）** — 物理世界、高风险决策、信任关系
2. **杠杆（中期）** — 从执行者变成指挥官，用 AI 放大自己
3. **资产（长期）** — AI 股权、稀缺资源、社会资本、人类认证溢价

### 品牌忠诚度对机器不存在
- AI Agent 只会优化，不管是谁家的服务
- 中介平台（外卖、打车、支付）会被 Agent 消除
