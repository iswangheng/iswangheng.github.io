## 🎯 具体技巧

### 1. 标题改写

**原文：** "10 questions (and answers) about how AI Gmail will affect your email outreach"

**改写：**
- "AI Gmail 如何影响销售触达？10 个关键问题解答"
- "Gemini 入驻 Gmail：销售人员需要知道的 10 件事"
- "2026 年的 Gmail 变革：10 个销售触达问题详解"

### 2. 段落改写

**原文：**
> "Google is turning Gmail into an AI-assisted inbox: it can summarize threads, help recipients search their inbox in natural language, and increasingly prioritize 'important' relationships over generic outreach."

**改写：**
> "Gmail 正在经历一次重大变革，Google 为其注入了 AI 能力。这意味着收件人可以快速总结邮件线程，用自然语言搜索历史记录，并且更倾向于关注重要联系人而非泛泛的营销邮件。"

### 3. 结构重组

**原文顺序：** 问题 1 → 问题 2 → 问题 3

**改写顺序：** 按主题分组
- 策略与内容类问题（1-3）
- 技术与送达率类问题（4-6）
- AI 体验类问题（7-10）

### 4. 数据重新表达

**原文：**
> "Our data suggests keeping emails under 100 words remains the sweet spot for replies."

**改写：**
> "根据 LeadContact 对数千封销售邮件的分析，字数控制在 100 以内的邮件获得回复的概率最高。简短、精炼的内容更容易被快速理解和回复。"

### 5. 注入 LeadContact 观点

**原文：** 通用建议

**改写：**
> "使用 LeadContact 的 Chrome 扩展，你可以在 LinkedIn 上实时查看验证后的联系信息，确保你发送邮件的对象准确无误。98% 的准确率让你的每封邮件都能到达对的人。"

## 🚫 避免的做法

### ❌ 不要这样做

1. **简单替换**
   ```javascript
   // 错误示例
   content.replace('Hunter', 'LeadContact')
   ```

2. **复制粘贴**
   ```javascript
   // 错误示例
   const newContent = originalContent + "\n\nTry LeadContact!"
   ```

3. **保留原文结构**
   ```markdown
   # 原标题
   [完全相同的段落内容]
   ```

### ✅ 应该这样做

1. **完全重写**
   ```javascript
   // 正确示例
   const newTitle = createNewTitle(originalTitle);
   const newContent = rewriteFromScratch(originalContent);
   ```

2. **重新组织**
   ```markdown
   # 全新的标题
   ## 重组的章节
   - 全新的观点列表
   ```

3. **添加独特价值**
   ```markdown
   ## LeadContact 的独家见解
   - 98% 准确率
   - Chrome 扩展集成
   ```

## 📊 重复度控制

### 目标重复度：< 30%

**如何降低重复度：**

1. **同义词替换**
   - "email outreach" → "销售触达", "商务邮件"
   - "cold email" → "陌生邮件", "开发信"
   - "deliverability" → "送达率", "投递效果"

2. **句式变换**
   - "A affects B" → "B 受到 A 的影响"
   - "It is important to" → "关键在于"
   - "This means that" → "这意味着"

3. **段落重组**
   - 调换段落顺序
   - 合并相似观点
   - 拆分长段落

4. **增加新内容**
   - 添加 LeadContact 数据
   - 引入实际案例
   - 提供具体工具使用方法
