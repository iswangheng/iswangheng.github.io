# LeadContact (LC) 产品深度理解报告

## 🎯 产品定位

**LeadContact** 是一个**B2B 销售触达工具**，帮助销售和招聘团队快速找到企业联系人和关键决策者。

### 核心价值主张
```
"Find Any Email, Find Any Phone Number"
（查找任何邮箱，查找任何电话号码）
```

**差异化卖点：**
- 🎯 **98% 准确率** - 高精度数据验证
- 💰 **更低的成本** - 集成多个数据源，无需单独付费
- 🤖 **AI 驱动** - 智能填充缺失的邮箱地址
- 🔗 **Chrome 扩展** - 无缝集成到工作流程

---

## 💡 核心功能

### 1. 📧 邮箱查找 (Email Finder)
- **120M+ 邮箱已找到**
- 访问经过验证的、最新的邮箱
- 即时获取结果
- 跨多个顶级数据源验证

### 2. 📞 电话号码查找 (Phone Number Finder)
- **60M+ 电话已提供**
- 跨多个顶级来源交叉验证电话数据
- 确保电话号码的有效性

### 3. 👔 决策者查找 (Decision Maker Finder)
- **270M+ 决策者已识别**
- 按职位头衔快速找到决策者
- 任何公司的决策者信息

### 4. 🔍 多场景触达

#### 场景 1: LinkedIn 个人资料页
- 在查看任何 LinkedIn 个人资料时使用 Chrome 扩展
- 一键显示验证后的联系信息

#### 场景 2: LinkedIn 公司页面
- 直接在公司页面上查找决策者和联系信息

#### 场景 3: LinkedIn 搜索结果
- 从 LinkedIn 搜索结果中批量查找验证后的邮箱和电话

#### 场景 4: 任何网站
- 浏览任何公司网站
- Chrome 扩展即时显示决策者和联系详情
